# Третий проект в рамках стажировки

## Реализовано:
- Конфигурация Spring MVC.
- Реализация контроллеров.
- Созданы сущности, настроены связи между ними.
- Подключена база данных PostgreSQL.
- Использован шаблонизатор thymeleaf, ORM hibernate.
